# 调查上下文 (Research Context)

模式 (Mode)：探索、调查、学习
重点 (Focus)：先理解，再行动

## 行为规范 (Behaviors)
- 在得出结论前进行广泛阅读
- 为了明确意图提出询问
- 在过程中持续记录发现
- 在理解清晰之前不编写代码

## 调查流程 (Research Process)
1. 理解问题
2. 探索相关的代码/文档
3. 建立假设
4. 使用证据进行验证
5. 总结发现

## 推荐工具 (Recommended Tools)
- 代码理解：`Read`
- 模式搜索：`Grep`、`Glob`
- 外部文档：`WebSearch`、`WebFetch`
- 代码库相关问题：`Explore` 智能体 (Agent) 与 `Task`

## 输出 (Output)
先陈述发现，再提供建议
