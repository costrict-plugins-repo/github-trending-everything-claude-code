# 构建修复 (Build Fix)

逐步修复 TypeScript 及构建错误（Build Errors）：

1. 执行构建：`npm run build` 或 `pnpm build`

2. 解析错误输出：
   * 按文件分组
   * 按严重程度排序

3. 针对每个错误：
   * 显示错误上下文（前后 5 行）
   * 说明问题
   * 提出修复建议
   * 应用修复
   * 再次执行构建
   * 验证错误是否已解决

4. 在以下情况下停止：
   * 修复导致了新的错误
   * 尝试 3 次后同一错误仍然存在
   * 用户请求暂停

5. 显示摘要：
   * 已修复的错误
   * 剩余的错误
   * 新引入的错误

为了安全起见，一次请仅修复一个错误！
